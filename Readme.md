# Практична робота №4 (Task4Variant2)

Цей проект містить реалізацію практичних завдань з програмування на мові C++ (стандарт C++20), орієнтованих на роботу з одновимірними масивами.

## Архітектура та система модулів

Проект побудований за модульним принципом, що дозволяє легко додавати нові завдання без зміни основного коду програми.
- **Головний файл (`main.cpp`)**: Відповідає лише за інтерфейс користувача (меню) та запуск обраних модулів.
- **Модулі (`modules/scripts/`)**: Кожне завдання винесене в окремий `.cpp` файл.
- **Автоматична реєстрація**: Використовується механізм самореєстрації через статичні об'єкти (`Modules.h`, `Modules.cpp`). При компіляції кожен модуль автоматично додається до загального списку, який використовує `main.cpp`. Це дозволяє розширювати функціонал, просто додаючи нові файли, не редагуючи ядро програми.

## Вимоги

- Компілятор C++ з підтримкою стандарту C++20 (MSVC, GCC, Clang).
- Система збірки CMake.

## Опис завдань

Усі завдання передбачають введення даних користувачем для забезпечення інтерактивності.

### Завдання 1 (`module1.cpp`)
**Робота з одновимірним масивом дійсних чисел**
Програма виконує дві операції над масивами, які вводить користувач.
- **Частина 1**: Для масиву дійсних чисел обчислюється:
  - Сума всіх від’ємних елементів.
  - Добуток елементів, що розташовані між мінімальним та максимальним елементами.
- **Частина 2**: Для масиву цілих чисел визначається максимальне число, яке зустрічається в масиві більше одного разу.

### Завдання 9 (`module2.cpp`)
**Аналіз та перетворення масиву**
Програма виконує комплексний аналіз та модифікацію масиву.
- **Частина 1**: Для масиву дійсних чисел виконуються наступні дії:
  - Знаходиться максимальний за модулем елемент.
  - Обчислюється сума елементів, розташованих між першим і другим додатними елементами.
  - Масив перетворюється таким чином, щоб усі елементи, що дорівнюють нулю та одиниці, були переміщені в кінець, зберігаючи відносний порядок інших елементів.
- **Частина 2**: Для масиву дійсних чисел знаходиться середнє арифметичне тих елементів, значення яких менші за задану користувачем величину.

## Збірка та запуск

1. Сконфігуруйте проект за допомогою CMake:
   ```bash
   cmake -S . -B build
   ```
2. Зберіть проект та запустіть створений виконуваний файл `task4variant2.exe`.
