# Практична робота №3 (Task3)

Цей проект містить реалізацію практичних завдань з програмування на мові C++ (стандарт C++20).

## Архітектура та система модулів

Проект побудований за модульним принципом, що дозволяє легко додавати нові завдання без зміни основного коду програми.
- **Головний файл (`main.cpp`)**: Відповідає лише за інтерфейс користувача (меню) та запуск обраних модулів.
- **Модулі (`modules/scripts/`)**: Кожне завдання винесене в окремий `.cpp` файл.
- **Автоматична реєстрація**: Використовується механізм самореєстрації через статичні об'єкти. При компіляції кожен модуль автоматично додається до загального списку, який використовує `main.cpp`. Це дозволяє розширювати функціонал, просто додаючи нові файли, не редагуючи ядро програми.

## Вимоги

- Компілятор C++ з підтримкою стандарту C++20 (MSVC, GCC, Clang).
- Система збірки CMake.

## Опис завдань

Усі завдання реалізовані двома методами: з використанням циклу **`for`** та циклу **`while`**.

### Завдання 1 (`module1.cpp`)
**Множення натуральних чисел**
Програма обчислює добуток двох натуральних чисел $m$ і $n$, використовуючи виключно операцію додавання.
- **Реалізація**: Алгоритм виконано двома способами: через цикл `for` та через цикл `while`.

### Завдання 2 (`module2.cpp`)
**Тризначні числа із заданою сумою цифр**
Програма визначає кількість тризначних натуральних чисел, сума цифр яких дорівнює введеному числу $n$ ($n \ge 1$).
- Перевірка коректності вводу (сума цифр для тризначного числа має бути в межах від 1 до 27).
- **Реалізація**: Перебір варіантів реалізовано через вкладені цикли `for` та `while`.

### Завдання 4 (`module4.cpp`)
**Статистика послідовності чисел**
Для заданого натурального числа $n$ та послідовності цілих чисел $a_1, a_2, \dots, a_n$ програма знаходить кількість:
- а) парних чисел;
- б) повних квадратів;
- в) квадратів непарних чисел.
- **Реалізація**: Обробка масиву та підрахунок виконано окремо через цикл `for` та цикл `while`.

### Завдання 5 (`module5.cpp`)
**Пошук Min/Max за умовами**
Для послідовності дійсних чисел програма знаходить екстремуми за різними критеріями:
- а) Мінімальне значення серед усіх елементів.
- б) Максимальне значення за модулем.
- в) Максимальне значення серед елементів на парних позиціях ($a_2, a_4, \dots$).
- г) Мінімальне значення серед елементів на непарних позиціях ($a_1, a_3, \dots$).
- ґ) Суму мінімуму на парних та максимуму на непарних позиціях.
- д) Максимальне значення серед елементів з індексами, що є степенями двійки ($a_1, a_2, a_4, a_8, \dots$).
- **Реалізація**: Пошук значень виконано у двох варіантах: з використанням циклу `for` та циклу `while`.

## Збірка та запуск

   ```bash
   cmake -S . -B build
   ```
2. Зберіть проект та запустіть створений виконуваний файл `task3`.
