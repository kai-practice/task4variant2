# Практична робота №4 (Task4Variant2)

Цей проект містить реалізацію практичних завдань з програмування на мові C++ (стандарт C++20), орієнтованих на роботу з одновимірними динамічними масивами.

## Архітектура та система модулів

Проект побудований за модульним принципом, що дозволяє легко додавати нові завдання без зміни основного коду програми.
- **Головний файл (`main.cpp`)**: Відповідає лише за інтерфейс користувача (меню) та запуск обраних модулів.
- **Модулі (`modules/scripts/`)**: Кожне завдання винесене в окремий `.cpp` файл.
- **Автоматична реєстрація**: Використовується механізм самореєстрації через статичні об'єкти (`Modules.h`, `Modules.cpp`). При компіляції кожен модуль автоматично додається до загального списку, який використовує `main.cpp`. Це дозволяє розширювати функціонал, просто додаючи нові файли, не редагуючи ядро програми.

## Вимоги

- Компілятор C++ з підтримкою стандарту C++20 (MSVC, GCC, Clang).
- Система збірки CMake.

## Опис завдань

Усі завдання передбачають гнучкий ввід даних: користувач може ввести значення вручну або залишити поле пустим (чи ввести некоректні дані) для автоматичної генерації випадкових чисел. Використовуються динамічні масиви (`new`/`delete`).

### Завдання 1 (`module1.cpp`)
**Робота з одновимірним масивом дійсних чисел**
Програма запитує розмір масиву та його елементи, після чого виконує:
1. **Обчислення суми від’ємних елементів**: Програма проходить по масиву та сумує всі числа менше нуля.
2. **Добуток елементів між Min та Max**: Знаходяться індекси мінімального та максимального елементів. Обчислюється добуток чисел, що розташовані між ними (не включаючи самі межі).
3. **Пошук дублікатів**: Визначається максимальне число, яке зустрічається в масиві більше одного разу.

### Завдання 9 (`module2.cpp`)
**Аналіз та перетворення масиву**
Програма працює з динамічним масивом, розмір та елементи якого можуть бути задані користувачем або згенеровані випадково.
1. **Максимальний за модулем елемент**: Знаходиться елемент з найбільшим абсолютним значенням.
2. **Сума між додатними**: Обчислюється сума елементів, розташованих між першим і другим додатними числами в масиві.
3. **Перетворення масиву**: Всі елементи, рівні `0` та `1`, переміщуються в кінець масиву, зберігаючи порядок інших чисел.
4. **Середнє арифметичне**: Обчислюється середнє значення елементів, які менші за задану користувачем величину (поріг).

## Збірка та запуск

1. Сконфігуруйте проект за допомогою CMake:
   ```bash
   cmake -S . -B build
   ```
2. Зберіть проект та запустіть створений виконуваний файл `task4variant2.exe`.
